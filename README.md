# CODE_pick
Приложение для отметки пиков/первых вступлений в SEG-Y файлах
*Анализатор SEG-Y файлов. Отметка пиков — Функционал*
**Основной стек инструментов:** Python (NumPy, SciPy, matplotlib, segyio, tkinter).

**Загрузка и подготовка данных**

Программа открывает сейсмические файлы формата SEG-Y, автоматически
считывает все трассы и временные отсчеты. Пользователь выбирает номер
трассы для анализа и задает временное окно просмотра в миллисекундах.
Интерфейс обновляется интерактивно при изменении параметров.

**Режимы детектирования**

**Режим отметки пиков.** Обнаруживает локальные максимумы на трассе.
Пользователь задает порог амплитуды для отсеивания шумов и минимальное
расстояние между пиками в отсчетах. Режим применяется для анализа
периодичности сигнала и выделения отражающих границ.

**Режим отметки времен вступления.** Фиксирует моменты превышения
сигналом заданного порога. После каждого срабатывания детектор
блокируется на указанный интервал времени, что позволяет выделять первые
вступления волн и разделять цуги импульсов.

**Визуализация**

Отображает три типа графиков. На графике трассы показываются
положительные и отрицательные полуволны, обнаруженные пики маркируются
красными крестами. Опционально строится огибающая сигнала методом
кубической интерполяции. График частот показывает зависимость мгновенной
частоты следования импульсов от времени. График амплитуд визуализирует
затухание сигнала в точках пиков.

**Расчет параметров**

Для каждой обнаруженной точки определяются время в миллисекундах и
амплитуда. Вычисляются интервалы между соседними событиями и
соответствующие им частоты в герцах. Рассчитываются максимальная и
минимальная амплитуды трассы.

**Экспорт результатов**

Формирует текстовый отчет, содержащий все времена пиков, интервалы,
частоты и амплитуды. Дополнительно реализован экспорт бинарной разметки:
времена событий преобразуются в последовательность нулей и единиц с
заданным пользователем шагом дискретизации. Полученный файл может
использоваться как размеченная выборка для обучения нейросетевых
моделей.
